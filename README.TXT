Description This is a memory matching game built with HTML, CSS, and JavaScript. The game includes a score tracker and level-up mechanics. Players can match pairs of cards and earn points. The game also includes a timer and tracks the number of flips.
Features
•	Memory matching game with pairs of cards
•	Score tracker to keep track of points
•	Timer to track the duration of the game
•	Flip counter to count the number of card flips
•	Level-up mechanics based on performance
•	Favorite company selection for double points
•	Quiz to test knowledge about the companies' creators
Rules
1.	Objective: Match all pairs of cards to win the game.
2.	Scoring:
•	Correct match: +10 points
•	Correct match of a favorite company: +20 points
3.	Timer: The game timer starts when the game begins.
4.	Flips: Each card flip counts towards the flip counter.
5.	Quiz: After each match, a quiz appears to test knowledge about the company's creator.
6.	Winning: The game is won when all pairs are matched.
Installation To run this project locally, follow these steps:
1.	Clone the Repository:
git clone https://github.com/DHRUVIT2312PARMAR/Memory-matching-game-with-score-tracker-and-level-up.git
2.	Navigate to the Project Directory:
cd Memory-matching-game-with-score-tracker-and-level-up
3.	Install Dependencies (if any):
npm install
4.	Start the Development Server (if using a local server):
npm start
Usage
1.	Open the Game:
•	Open index.html in your web browser to start the game.
•	Alternatively, if you are using a local server, navigate to the server's URL in your browser.
2.	Select Your Favorite Company:
•	Before starting the game, select your favorite company from the dropdown menu.
3.	Start the Game:
•	Click the "Start Game" button to begin.
4.	Play the Game:
•	Flip cards to find matching pairs.
•	Earn points for each correct match.
•	Answer the quiz questions to test your knowledge.
5.	Win the Game:
•	Match all pairs to win the game.
•	Your score, time, and flips will be displayed.
